Project Overview

"Lazer Security System(2023)" The objective of the "Laser Security System" project is to design and implement an advanced and highly affordable security solution using a laser and Light Dependent Resistor (LDR). The system aims to detect disturbances in the laser beam caused by intruders or objects, triggering alarms or activating surveillance measures.

Voice Controlled Car

"Voice Controlled Car (2023)" The objective of the "Voice Controlled Car" project is to design and implement a car control system that responds to voice commands. This innovative project allows users to control a car's movement using their voice, enhancing the convenience and accessibility of remote control systems.
